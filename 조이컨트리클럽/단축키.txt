ctrl + a - 전체 선택
ctrl + c - 복사
ctrl + v - 붙여넣기
ctrl + d - 똑같은 텍스트를 동시에 선택
ctrl + s - 다들 알겠지만 저장 (자주 사용하기!!)
ctrl + alt + 방향키(위, 아래) - 방향키 방향으로 커서 여러개 생성 (여러줄에 동시에 입력할 때 사용)
ctrl + shift + 방향키(왼쪽, 오른쪽) - 단어 단위로 텍스트 선택
alt + shift + f - 코드 자동정렬 (자주 사용하기!!)
shift + 방향키 - 방향키 방향으로 선택
shift + end - 커서부터 해당 줄의 끝부분까지 선택
shift + home - 커서부터 해당 줄의 첫부분까지 선택
end - 해당 줄의 끝부분으로 커서 이동
home - 해당 줄의 앞부분으로 커서 이동
f2 - 파일이나 폴더명 변경
delete - 파일 삭제 혹은 커서 뒷부분 한글자 제거
